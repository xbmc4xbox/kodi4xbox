## Kodi
This is port of XBMC 4.0+ and Kodi v20 Nexus to Original Xbox using open source toolchain known as [NXDK (New Xbox Development Kit)](https://github.com/XboxDev/nxdk). It's currently in pre-alpha stage and can only be compiled. I will need help from other devs to get it working, so contributions are welcome!

## Current status
Project is compiling and when launched we splash screen and Home window are rendering properly. Main render loop is running without any crashes. Currently there is no support for joystick inputs, audio, video and music playback etc. For more detailes, please check our [roadmap](docs/roadmap/README.md).
